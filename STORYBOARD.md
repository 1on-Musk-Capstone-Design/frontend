# ON-IT 시연 영상 스토리보드 & 콘티

## 📋 영상 개요

**제목**: "ON-IT - 아이디어를 자유롭게, 함께 그려보세요"
**길이**: 60초 (광고 영상 스타일)
**타겟**: 협업이 필요한 모든 팀 (개발팀, 기획팀, 디자인팀, 교육기관, 해커톤 팀 등)
**톤앤매너**: 현대적, 역동적, 감성적, 혁신적

## 🎬 시퀀스별 상세 스토리보드

### **시퀀스 1: 오프닝 (0:00-0:05)**

**화면 구성:**
- **컷 1-1 (0:00-0:02)**: 검은 화면에서 "ON-IT" 로고가 페이드인
  - 로고: 깔끔한 타이포그래피, 초록색(#01CD15) 강조
  - 배경: 미니멀한 그라데이션 (검정 → 어두운 회색)
  - 효과: 부드러운 페이드인 + 약간의 스케일 애니메이션

- **컷 1-2 (0:02-0:05)**: 로고가 사라지며 캔버스가 확장되는 애니메이션
  - 격자 무한 캔버스가 화면 중앙에서 확장되며 나타남
  - 줌 아웃 효과로 무한 공간 느낌 강조
  - 배경음: 미래지향적인 부드러운 전자음 (Ambient)

**나레이션/자막:**
- "아이디어는 무한합니다"

**카메라 워크:**
- 정적 샷 → 줌 아웃

### **시퀀스 4: 실시간 협업 (0:20-0:32)**

**화면 구성:**
- **컷 4-1 (0:20-0:24)**: 멀티뷰 - 여러 사용자가 동시에 작업하는 모습
  - 화면 분할: 4분할 화면으로 4명의 팀원이 동시에 작업
  - 각 화면에서 서로 다른 아이디어 카드를 생성하고 배치
  - 실시간으로 변경사항이 동기화되는 모습
  - 각 사용자의 커서가 다른 색상으로 표시됨

- **컷 4-2 (0:24-0:28)**: 채팅 기능 시연

  - "좋은 아이디어네요!" "이 부분 더 자세히 설명해주세요" 등의 대화
  - 메시지에 위치 링크가 포함되어 클릭하면 해당 아이디어로 이동

- **컷 4-3 (0:28-0:32)**: 참가자 알림
  - 상단 툴바에 참가자 아바타가 실시간으로 추가됨
  - "김민수님이 참가했습니다" 알림
  - 여러 사람의 아바타가 동시에 표시됨

**나레이션/자막:**
- "실시간으로 함께 작업하세요"
- "채팅으로 즉시 소통하고, 참가자의 작업을 실시간으로 확인하세요"

**카메라 워크:**

## 🎨 비주얼 스타일 가이드

### **색상 팔레트**
- **Primary**: #01CD15 (밝은 초록)
- **Background**: #FFFFFF (화이트)
- **Text**: #111827 (다크 그레이)
- **Accent**: #F3F4F6 (라이트 그레이)
- **Error/Warning**: #F59E0B (오렌지)

### **타이포그래피**
- **제목**: Bold, Sans-serif (Pretendard 또는 Noto Sans)
- **본문**: Regular, Sans-serif
- **로고**: Custom, Modern

### **애니메이션 스타일**
- 부드러운 이징 (ease-in-out)
- 빠른 전환 (0.3초 이내)
- 스케일, 페이드, 슬라이드 효과 활용

### **음악/사운드**
- **오프닝**: 미래지향적인 Ambient 음악
- **중간**: 업템포 전자음악 (협업의 역동성 표현)
- **클로징**: 밝고 희망적인 톤
- **효과음**: 
  - 클릭 사운드 (부드러운 톤)
  - 성공 사운드 (클러스터링 완료 시)
  - 알림 사운드 (참가자 입장 시)

4. **간편한 접근**: 초대 링크 한 번으로 누구나 쉽게 참여하세요
5. **모든 협업에**: 개발팀부터 해커톤까지, 모든 팀에 적합합니다

## 🎬 촬영/제작 노트

1. **화면 녹화 시 주의사항**
   - 고해상도로 녹화 (최소 1920x1080, 권장 4K)
   - 부드러운 마우스 움직임 (마우스 가속도 조절)
   - 깔끔한 UI (불필요한 요소 숨기기)
   - 실제 사용자처럼 자연스러운 인터랙션

2. **편집 시 주의사항**
   - 빠른 전환으로 역동성 표현
   - 부드러운 이징으로 전문성 표현
   - 색감 일관성 유지
   - 자막 가독성 확보

3. **음악 선택 기준**
   - 저작권 없는 음악 사용 (예: YouTube Audio Library, Epidemic Sound)
   - 서비스 톤과 일치하는 음악
   - 볼륨 조절 (나레이션/효과음과의 균형)
